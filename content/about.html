ОСНОВНЫЕ ФУНКЦИИ VMS :
        1. Импорт данных


        • Импортирование из файла - быстрый способ добавления шпона в вашу Базу с помощью таблицы данных полученных у поставщика;
        • Запись вручную - оприходование  шпона и других материалов с помощью сканера;
        • Паллета - подробная информация по отдельной паллете: количество, список колод  и пачек, все размеры и др.
        • Состояние склада - отображение склада по каждой породе или целиком, наличие фильтра по длине и ширине с процентами;
        • Перемещение по складу - возможность перемещения отдельной паллеты или остатков на другую;

        2. Продажа

                • Новый заказ (ОПТ) - реализация большого количества шпона (паллета / машина);
                • Новый заказ (Розница) - продажа небольшого количества шпона и других материалов;
                • Обзор заказов - отображение текущих заказов и управление ими;

        3. Сервис

                • Дополнительная информация - добавление клиентов, характеристик материаллов и др.;
                • Работа со штрих-кодами - настройка штрих-кодов другого производителя;
                • Авторизация - настройка пользователей системы и прав доступа.


        Данный функционал является базовым, но исходя из особенностей работы каждого предприятия возможны внесение изменений согласно
        требований заказчика. Оплата и сроки их исполнения по договоренности.
